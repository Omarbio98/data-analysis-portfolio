# Behavioral Analysis of *Perdita maritima*

**External collaboration with Dr. Bryan N. Danforth**
2024 - Present

## 🎯 Objective
To evaluate behavioral patterns of a solitary native bee by integrating fixed and random effects variables.

## 📊 Methodology
- Generalized Mixed Models (GLMM) using the lme4 package
- Processing of observational data on animal behavior
- Generation of graphs for visualization and interpretation of results

## 🛠 Tools
R, RStudio, packages: lme4, ggplot2, tidyverse

## 📂 Folder Contents
- `scripts/`: Documented and reproducible R code; results available after publication.

Results in preparation for publication in an indexed journal.
